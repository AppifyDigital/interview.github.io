Overview

Build a full-stack web application using Next.js for the frontend and Nest.js for the backend, deployed on Railway.app. You have 48 hours to complete this assignment.

Requirements Checklist

1. Project Structure

Frontend (Next.js):

Backend (Nest.js):

2. Frontend Development

Technologies:

  • Use Next.js with TypeScript.

Features:

  • Routing: Implement navigation between pages.
  • API Integration: Integrate with backend APIs to fetch and manipulate data.

User Interfaces:

  • Login Page:
    • Implement JWT authentication.
  • CRUD Interface:

    Choose one of the following entities to perform Create, Read, Update, and Delete operations:

Design:

3. Backend Development

Technologies:

  • Use Nest.js with TypeScript.

Features:

  • Implement middleware for logging and error handling.

Database:

  • Integrate Prisma as the ORM.
  • Implement data validations and secure data handling.

4. Authentication

  • Implement JWT Authentication to secure API endpoints.
  • Protect frontend routes to ensure only authenticated users can access certain pages.

5. CRUD Operations

  • Develop Create, Read, Update, and Delete functionalities for the chosen entity.
  • Implement proper validation and error handling for all operations.

6. Deployment

  • Deploy the application on Railway.app.
  • Provide instructions to access and test the deployed app.
